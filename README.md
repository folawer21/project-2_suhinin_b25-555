# Primitive Database

Простая база данных с интерфейсом командной строки.

## Установка

```bash
pip install project-2-suhinin-b25-555

### Обработка ошибок
Все операции базы данных теперь защищены от ошибок. Система автоматически обрабатывает:
- Отсутствие файлов данных
- Обращение к несуществующим таблицам и столбцам  
- Ошибки валидации данных

### Подтверждение действий
Опасные операции требуют подтверждения:
- Удаление таблиц (`drop_table`)
- Удаление записей (`delete`)

### Производительность
- Замер времени выполнения операций
- Кэширование результатов запросов для ускорения повторных выборок

### Логирование
Автоматическое логирование времени выполнения операций работы с данными.
